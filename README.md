🚀 Elementum Frontend Assignment

A responsive React-based landing page built from a Figma design, focusing on clean UI implementation, component structure, and responsiveness.

---

🧠 Overview

This project is a pixel-accurate implementation of a modern landing page using React and Tailwind CSS.
The goal was to convert a Figma design into a functional, responsive web application with clean and reusable code.

---

✨ Features

- 📱 Fully responsive (Mobile, Tablet, Desktop)
- 🎯 Close-to-Figma UI implementation
- 🧩 Reusable React components
- 🎨 Styled using Tailwind CSS
- ⚡ Smooth layout and spacing consistency
- 🐳 Docker support for easy setup

---

🛠️ Tech Stack

- React (Vite)
- Tailwind CSS
- Docker

---

📂 Project Structure

```
src/

```


---

🐳 Run with Docker

    1. Build the image

    docker build -t elementum-frontend .

    2. Run the container

    docker run -p 5173:5173 elementum-frontend

    3. Open in browser

     http://localhost:5173

---

💻 Run Locally

    Install dependencies

    npm install

    Start development server

    npm run dev
